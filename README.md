# Deepfake Detection via Multi-Layer Local Intrinsic Dimensionality

This experimental project detects deepfake images and videos by measuring how the **local intrinsic dimension (LID)** of feature representations changes across the depth of a Vision Transformer — without retraining the backbone.

## Why LID?

Most deepfake detectors fine-tune a classifier directly on top of visual features. This project takes a different angle: rather than asking *what* a face looks like, it asks *how the geometry of its representations is structured* inside the model.

Deepfake images generated by GANs or diffusion models occupy a different manifold than real photographs. That difference is measurable as a shift in local intrinsic dimension — the effective dimensionality of the neighborhood around each sample's embedding. Real images tend to cluster in higher-dimensional, more complex manifolds; generated images collapse to lower-dimensional, more regular ones.

## Approach

LID is estimated using the **TwoNN method** (Facco et al., 2017), which derives a dimension estimate from the ratio of the 1st and 2nd nearest-neighbor distances under a Pareto model. The scalar estimate is extended to a per-point *k*-NN log-ratio feature vector, which a shallow MLP can then learn to discriminate.

Crucially, LID is computed at **three depths** of the ViT encoder:

```
ViT (frozen)  →  start / middle / end layer CLS tokens
                        ↓ LID features (k-dim each)
              concat (3k-dim) → MLP → Real / Fake
```

The backbone (`prithivMLmods/Deep-Fake-Detector-v2-Model`) is fully frozen. Only the MLP head is trained for one epoch on `prithivMLmods/Deepfake-vs-Real-60K`.

## Differentiation

| Aspect | Typical deepfake detectors | This project |
|--------|---------------------------|--------------|
| What is learned | Visual artifacts, texture | Geometric structure of representations |
| Backbone usage | Fine-tuned end-to-end | Frozen; only geometry is extracted |
| Feature type | Class logits or pixel features | Intrinsic dimension across layers |
| Generalization hypothesis | Artifact-specific | Manifold-level, architecture-agnostic |

The bet is that LID captures a more fundamental signal than surface-level artifacts, and that this signal is more robust across deepfake generators the model has never seen.

## Workflow

```
train.py        fine-tune the MLP head (1 epoch, ViT frozen)
                → saves model/lid_classifier.pth

inference.py    load classifier + run on ./data/
                → face detection (dlib), video frame sampling
                → batched LID inference
                → submission.csv
```
